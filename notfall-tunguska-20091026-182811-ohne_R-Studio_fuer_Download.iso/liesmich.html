COMPUTERBILD Notfall-CD 2.0 — bootfähiges System

Auf dieser CD befindet sich ein komplettes Notfall-System, das unabhängig von Windows läuft. Falls Sie die Notfall-Werkzeuge jetzt benutzen wollen, starten Sie Ihren Computer neu. In den meisten Fällen startet Ihr PC dann automatisch von der CD. Falls nicht, stellen die Startreihenfolge so ein, dass der Computer zuerst vom CD-/DVD-Laufwerk hochfährt.

Mehr zu den Startoptionen und den Rettungs- und Wartungswerkzeugen lesen Sie im Artikel zur CD in COMPUTERBILD 24/2009.
